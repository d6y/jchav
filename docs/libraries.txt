To run JChav you need the following libraries:

1.  JFreeChart

    Url: http://www.jfree.org/jfreechart/index.html
    License: LGPL
    Version: 1.0.2
    JARs: jfreechart-1.0.2/lib/jfreechart-1.0.2.jar
    
2.  JCommon

	Url:  http://www.jfree.org/jcommon/index.php
	License: LGPL
	Version: 1.0.6
	JARs: jcommon-1.0.6/jcommon-1.0.6.jar
