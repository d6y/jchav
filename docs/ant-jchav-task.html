JChav

Description

The JChav ant task allows for charts to be generated as part of an automated ant build.

Parameters

The parameters on the JChav task are used to specify the input, and output folders to execute the task against. It is also possible to specify the global page titles, and whether or not to manage axes.

Attribute Description Required
srcDir The folder that contains the jmeter results that are to be charted.
NB: This attribute is deperecated, use the ant-friendly <FileSet> instead (see below)
This attribute will be removed shortly, probably in the next release. 		No
destDir The folder that JChav results should be written to. Yes
reporttitle The global title that should appear in the main report. No
uniformyaxis Should the Y-axis be rendered uniformly. No, default is false

Examples

<!-- Run the charting task over the jmeter results in srcdir. -->
<jchav srcdir="${jmeter.result.dir}" 
       destdir="${jchav.result.dir}"  
       reporttitle="Digg Example Walk"
       uniformyaxis="true"  />
